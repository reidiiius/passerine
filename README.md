# Passerine
Ornithology

---

### Download

    git clone https://github.com/reidiiius/passerine.git

---

### Usage
command-line

    cd passerine/

:

    tclsh syrinx.tcl

:

    tclsh syrinx.tcl help

:

    tclsh syrinx.tcl eadgbe n0 j3

search example

    tclsh syrinx.tcl 56

screenful view

    tclsh syrinx.tcl eadgbe flock | sensible-pager
